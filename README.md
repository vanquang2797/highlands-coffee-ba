| Item | Detail |

## 🔍 Key Deliverables

### Business Requirements Document (BRD)
- **120+ requirements** covering functional and non-functional areas
- Prioritized using **MoSCoW method** (Must/Should/Could/Won't)
- Structured per BABOK v3 knowledge areas: Needs Assessment, Stakeholder Engagement, Requirements Analysis

### Stakeholder Analysis
- Identified 6+ stakeholder groups (customers, store staff, marketing team, IT, management)
- Mapped using Interest vs. Influence matrix
- Engagement strategy defined per group

### Use Case Modeling
- Core use cases: Browse menu, Place order, Earn/Redeem loyalty points, View order history
- Includes: primary actors, pre/post conditions, main flow, alternative flows
